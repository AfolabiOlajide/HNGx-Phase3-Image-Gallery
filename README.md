## Running the code on your local machine
- download zip folder or clone the application
```
git clone https://github.com/AfolabiOlajide/HNGx-Phase3-Image-Gallery.git
```
- install the application
```
yarn add
```
or
```
npm install
```
- run the application
```
yarn run dev
```
or
```
npm run dev
```
